# Footer NOVA51

Estructura profesional del componente footer para la plataforma de rifas NOVA51.

## Estructura de la carpeta

- `footer.html`  → Estructura HTML principal del footer
- `footer.css`   → Estilos CSS para el diseño y animaciones
- `footer.js`    → Funcionalidades interactivas y scripts
- `README.md`    → Documentación y guía de integración

## Integración

1. Incluye el HTML en tu archivo principal (`index.html`).
2. Vincula el CSS en el `<head>`:
   ```html
   <link rel="stylesheet" href="footer/footer.css">
   ```
3. Vincula el JS antes de `</body>`:
   ```html
   <script src="footer/footer.js"></script>
   ```

## Personalización
- Modifica los enlaces, imágenes y estilos según la identidad de tu proyecto.
- Los nombres de los desarrolladores tienen animación y enlaces directos a Instagram.
